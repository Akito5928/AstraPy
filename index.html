Why Python?

日経クロステックの「プログラミング言語利用実態調査2024」では、 Python が 42.7% で 2 年連続の 1 位となっています。 教育・AI・データ分析・業務自動化など、幅広い分野で最も使われている言語です。

AstraPy は、この “最も使われている言語” をブロックで扱えるようにし、 誰でも Python の世界に入れるようにする NexFlow プロジェクトの第一歩です。

About NexFlow

NexFlow は「次の流れを作る」ことを目指すプロジェクトです。 ブロックで本物のプログラミングを扱い、将来的には 3D Scratch や Unity を Python で動かす中間層エンジンへと発展します。
